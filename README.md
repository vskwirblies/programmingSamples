## Java
Beebop is a small program that prints numbers from 0 to a given limit.
While doing so, it replaces certain numbers with 'bee', 'bop' and 'beebop'.
To be more specific:
- numbers that are divisible by three get replaced with 'bee',
- numbers that are divisible by five get replaced with 'bop',
- numbers that are divisible by both three and five get replaced with 'beebop'.
